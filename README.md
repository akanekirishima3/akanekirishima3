### SS27　友情について  Of Friendship (1612, rewritten 1625)
~~~
https://en.wikisource.org/wiki/The_Works_of_Francis_Bacon,_Volume_1/Essays/Of_Friendship


　「孤独を喜ぶ者は誰でも野獣か神である」と言った人(†1)でも、この言葉におけるほど、
真理と虚偽の両方を数語で言い表わすことは難しかったであろう。
なぜなら、社会に対して生まれつきひそかに憎悪と嫌悪を抱くことには、どんな人間にあっても野獣めいたところがある、ということはこの上ない真理であるが、
しかし、そのことに神の性質を思わせる特徴がいくらかある、ということはこの上ない虚偽だからである。
ただし、そうしたことが孤独を楽しむからではなく、
高尚な生活のために身を隠したい願望から起こるなら、話は別である。
カンディアのエピメニデス、ローマのヌマ、シチリアのエムペドクレス、テュアナのアポロニオスのような異教徒の一部には、
そのように偽り、そのようなふりをしたことが見られるが(†2)、古代の隠者や教会の神父の幾人かは、確かに実際にそうであった。
しかし人々は孤独が何であるか、
またどの程度までが孤独であるか、ほとんど気づいていない。
群衆は仲間ではなく、顔を並べただけでは絵の陳列室にすぎず、
愛のない所では、会話はシンバルをちりんちりん鳴らすにすぎないからである。
「大きな都市は大きな孤独(†3)」というラテン語の諺は、そのことを示唆している。
大きな都市では友人が散らばっているので、狭い地域にあるあの親しいつきあいが、多くの場合、ないからである。
しかし、われわれはさらに進んで、この上なく真実のこととして、真の友人をもたないことが、それこそ悲惨な孤独である、と断言することができる。
それがなければ、この世は荒野にすぎない。
そして、孤独のこのような意味から言っても、性質や感情が友情に不向きにできている人は誰でも、それを野獣から受けついでいるのであって、人間性から
友情の主な効果は、あらゆる種類の情念が原因となって引き起こす、心のわだかまりや高ぶりを和らげて発散させることである。
われわれは機能障害と息切れの病気が身体に最も危険であることを知っている。
心においても、それほど違ってはいない。
肝臓をよくするにはサルサ剤(†4)を用い、膵臓をよくするには鉄剤を、
肺には硫黄華、脳には|海狸香《かいりこう》を用いればよい(†5)。
しかし心を開くには、真の友人のほかに、どんな処方もない。
真の友人には悲しみも喜びも望みも疑いも忠告も、心に重くのしかかるどんなことでも、いわば俗人の告白もしくは懺悔として打ち明けることができる。

　偉大な国王や君主が、われわれの論じている友情の効果を、実に高く評価しているのを見るのは意外である。
彼らはしばしば自分の安全や王位を危険にさらしても、それを買い求めるほどである。
それと言うのも王侯は、自分の身分と臣下や召使のそれとがかけ離れているので、
この友情の効果を収めるには(自分にそれができるために)二、三の者を引き上げて、
いわば自分の仲間とし、ほとんど対等の者とせずにはいられないが、そうすることはしばしば面倒なことにもなるからである。
近代の言語はこういう人々に対して、寵臣とか側近とかの名称を与えている。
まるで恩恵か対談が問題であるかのようである。
ところが、ローマの名称はその真の用途と原因に触れて、彼らを「ともに心配する者」と呼んでいる。
そういうことが両者を結ぶ絆となるからである。
またわれわれは、こうしたことが無気力な、怒りやすい王侯によってばかりでなく、
かつて君臨した最も賢明で慎重な王侯によってもなされたことを明らかに知っている。
彼らは随従者の幾人かを同列に扱い、平民の間で受け入れられている言葉を用いて、
互いに友と呼び、他の人々にも同様に自分たちをそのように呼ぶことを許した。

　ルキウス・スラがローマを支配していた時、彼はポンペイウス(のちに偉大なという異名をつけられた)を高い地位に引き上げ、
その結果ポンペイウスは自分をスラにまさる者と高言するようになった。
というのは、彼がスラの思わくに反して、自分の一友人のために選挙運動をして執政官の地位を勝ち取った時、
スラがそのことにいささか|憤慨《ふんがい》して、|居丈高《いたけだか》になって話し始めると、
彼は再びスラにくってかかって、「沈む太陽より昇る太陽を拝む人のほうが多いのだから」と言い、実際にはスラにことを荒立てないように命じたからである(†6)。

　デキムス・ブルートゥスはユリウス・カエサルを動かす力をもつようになり、
カエサルはその遺言書の中でブルートゥスを自分の甥に次ぐ残余権者(†7)に指定したほどである。
それなのに、この男こそカエサルを思いのままにあしらって死へ引き渡した者である。
というのは、カエサルが不吉な前兆のために、
とくに〔妻の〕カルプルニアの夢のために、|元老《げんろう》院〔の会議〕を解散しようとした時、
この男はカエサルにカエサルの妻がもっとよい夢を見るまで、
|元老《げんろう》院〔の会議〕を延期しないように望むと言いながら、カエサルの腕を取って静かに椅子から立ち上らせたからである(†8)。
また彼がカエサルから受けた恩瀬は並々でなかったらしく、
アントニウスは彼のこと、クロの『フィリッボス演説』の一つで文字どおり引用されている手紙の中で、
あたかも彼がカエサルに魔法をかけたかのように、魔女と呼んでいるほどである(†9)。
アウグストゥスはアグリッパを(卑しい生まれであったけれども)非常な高位に引き上げたので、
アウグストゥスが自分の娘エリアの結婚のことでマエケナスに相談した時、マエケナスは無礼を顧みず
「陛下が皇女をアグリッパに嫁がせになるか、さもなければ彼の命を奪わなければなりません。
そのほかに道はありません。
陛下は彼をそれほど偉くされました」と答えた(†10)。

　ティベリウス・カエサルの場合には、セイアヌスが非常な高位にのぼり、二人は盟友と呼ばれ、またそう見なされたほどである。
ティベリウスはセイアヌスに与えた手紙の中で、
「こうしたことを私はわれわれの友情のために隠さなかった」と言っているし、
|元老《げんろう》院もこぞって二人のはなはだ親密な友情の故に、「友情」に対して、まるで女神に対するように、祭壇を捧げた(†11)。

　同じようなこと、あるいはそれ以上のことが、セプティミウス・セウェルスとプラウティアメスとの間にもあった。
というのは、セウェルスはその長子を無理にプラウティアヌスの娘と結婚させたが、
しばしばその長子をプラウティアヌスが侮辱するのを支持したし、また|元老《げんろう》院に与えた手紙の中で
「私はあの男を非常に愛しているから、彼が私より長生きすることを願っている」というような言葉を書き添えている(†12)。

　さて、これらの王侯がトラヤヌス(†13)とかマルクス・アウレリウスとかのような人であったならば、
こうしたことは天性の豊かな美質から起こったことだと考えられたかも知れない。
しかし、これらの人々はいずれもはなはだ賢明で、非常に強固なきびしい心をもつ人であり、また極端に自分自身を大事にする人であったから、
そのことから極めて明白に分かることは、彼らが自分自身の恵まれた境遇を(死を免れない人間に起こりうる最もすばらしいものであったけれども)、
友人を得て完全にすることができなければ、不完全なものにすぎないと思ったということである。
しかもそれにとどまらず、彼らは妻も子も甥もある王侯であったにもかかわらず、
これらの者が誰も友情の慰めを与えることができなかった、ということである。

　コミーヌ(†14)が最初の主君、シャルル勇猛公(†15)について述べていることは、忘れられてはならない。
すなわち公は自分の秘密を誰にも打ち明けようとしなかった、
とくに公を最も悩ませた秘密はそうであったと。
彼はそれについてつづけて「公の晩年には、そのような秘密主義が公の知力を損ない、少しばかり弱らせた」と言っている。
確かにコミーヌは、もし彼にその気があったならば、彼の第二の主君ルイ十一世(†16)についても、同じ判断を下すことができたであろう。
この王も秘密主義に苦しんだからである。
ピュタゴラスの「心を食べるな(†17)」という諺は、謎めいているが真実である。
確かに、それを堅苦しく言い表わそうとするならば、
本心を打ち明ける友人をもたない人々は、自分自身の心を食べる食人種である。
しかし、最も驚嘆すべきことが一つある(私はそれを述べてこの友情の効果を終えたいと思う)。
それは前述したように友人に本心を伝えることは、二つの相反する結果をもたらすということである。
それは喜びを二倍にし、悲しみを半分に切りつめるからである。
それと言うのも、喜びを友人に伝えて、喜びをさらに一段と大きくしない者はなく、悲しみを友人に伝えて、
悲しみをさらに一段と小さくしない者はないからである。
それゆえ、それが人の心に及ぼす作用は、錬金術師たちがいつも人体に利くと言っていた石の効能に似ている。
すなわち、それは全く相反する結果をもたらすが、つねに自然に役立ち、自然のためになる。
けれども、錬金術師の助けを求めるまでもなく、自然のありきたりの進行の中に、
このことを明白に示す具体例がある。
物体においては、結合がどんな自然的活動でも強化し助長するが、他方ではどんな強制的圧迫でも弱くし鈍くするからである(†18)。
心についても全く同じである。

　友情の第二の効果は、第一の効果が感情に対してそうであるように、知性に対して健全をもたらすものであり、最上のものである。
なるほど友情は感情にあっては、荒天と嵐を晴天にするが、しかし知性にあっては、思想の闇と混乱を白昼にするからである。
またこのことは、
人が友人から受ける誠意のこもった忠告についてだけだと思ってはならない。
忠告を受けるようになる以前でも、確かなことだが、心に多くの考えをもっている者は誰でも、
それを他の人に伝えたり話したりすることによって、才知や知性が明晰になり、ほぐれてくる。
そうなれば、彼は自分の思想をもっと容易に検討し、
もっと整然と並べるし、言葉に言い表わす時、どのように思われるかが分かる。
最後に、以前より賢くなる。
しかも、一日の瞑想より一時間の対話によってそうなる。
テミストクレスがペルシャ王にこう語ったのは至言である。
「談話はアラス織りの布をひろげてみせるようなものです。
そうすれば、想像された模様が図柄となって現われます。
ところが、考えられただけでは、それらはただ包まれているようなものです(†19)」。

　知性を啓発することにあるこの友情の第二の効果も、人に忠告することができるような友人だけに限られるわけではない。
(もちろん、そのような友人が最上ではあるが)、そのようなこ
なくても、みずから進んで学び、自分の考えをはっきり表現し、それ自身ではものを切らない砥石にあてて、自分の才知を磨くのである。
要するに、自分の考えを窒息させるくらいなら、
彫像か絵に話しかけるほうがましであろう。
さて、友情のこの第二の効果を完全にするために、もっとよく知られていて普通に見受けられるもう一つの点をつけ加えよう。
それは友人の誠意のこもった忠告である。
ヘラクレイトスはその謎めいた言葉の一つで、「乾いた光がつねに最もよい」とうまいことを言っている(†20)。
人が他人の忠告から受ける光が、自分自身の知性と判断力から発する光より、
乾いていて純粋であることは確かである。
後者の光はつねにその人の感情や習慣の中にとっぷり漬かっていて濡れている。
それゆえ、友人が与える忠告と人が自分自身に与える忠告との間には、友人の忠告と阿諛者の忠告との間にあるのと同じほど多くの相違がある。
自分自身のごとき阿諛者はいないからである。
また自分自身の阿諛に対しては、友人の無遠慮にまさる対策はない。

　忠告には二種類ある。
一つは態度に関するものであり、もう一つは仕事に関するものである。
第一のものについて言えば、心を健全に保つ予防薬は友人の誠意のこもった警告である。
自分困自身をきびしく追及することは、薬ではあるが、時として骨身にこたえすぎるし、消耗させすぎる。
道徳に関する良書を読むことは、いささか味気なくて退屈である。
自分の欠点を他人の
中に認めることは、時おり〔他人の欠点を正しく見ないので〕自分の場合に妥当しないことがある。
しかし最もよい処方(私の言うのは効能の点でも服用の点でも最もよい)は、友人の警告である。
多くの人(とくに偉い人々)が、教えてくれる友人をもたないために、
どんなに大きい過ちや途方もないばかげたことをしでかして、自分の名声や幸運をひどく傷つけるか、を見るのは意外である。
それと言うのも、聖ヤコブが言っているように、彼らは「時たま鏡をのぞきこむが、自分の姿や顔をすぐに忘れてしまう(†21)」ような人だからである。

　仕事について言えば、両眼が片目よりよく見えないだの、賭博者がつねに傍観者よりよく見ているだの、
腹を立てた人が怒る前にラテン語の二十四の字母を暗唱した人と同じように賢明であるだの、
銃は腕でささえても台でささえるのと同じようにうまく撃てるだの、と考えたい人は考えるがよいし、また自分自身を全能と思うために、
このような浅はかで高慢な想像をたくましくしてもよい。
しかし、どっちみち、仕事を順調に押し進めるのは、よい忠告の助けである。
また、もし誰かが忠告は受けたいが、少しずつにしたいと考え、ある仕事にはある人から、他の仕事には他の人から忠告を求めるならば、
それも結構である(少しも求めない場合よりは、おそらくよいだろうという意味である)。
しかし、そういう人は二つの危険を冒す。
一つは、誠意のこもった忠告は受けられないだろうということである。
本当に申し分のない誠実な友人からならいざ知らず、
忠告する人が抱いている何かの目的のために曲げられたり歪められたりしないような忠告を与えられることは、めったにないからである。
もう一つは、(善意からであるにしても)有害であてにならない、
害悪と対策とが半々に入りまじっている忠告を与えられかねないからである。
それはちょうど諸君が、諸君の訴える病気の治療にはあつらえ向きだと考えられているが、
諸君の身体のことは知らない医者を呼ぶようなものである。
すると、そのような医者は諸君をさしあたり快方に向かわせるかも知れないが、何かほかの方面で諸君の健康をすっかり害して、
病気は治すが患者は殺すことになるかも知れない。
ところが、こちらの事情を知りつくしている友人は、何か当座の仕事を促進しても、別の厄介なことにぶつからないように気をつけてくれるであろう。
それゆえ、てんでばらばらな忠告には頼らないことだ。
そのような忠告は決定したり指示を与えたりするより、むしろ注意をそらせたり迷わせたりするであろう。

　友情のこれら二つの貴重な効果(感情における平静と判断力のささえ)のあとに、最後の効果がつづく。
それは多くの粒がつまっているザクロに似ている。
私の言うのは、すべての行動と
好機における援助と関与である。
ここで友情のいろいろな効用を生き生きと思い描く最上の方法は、自分だけではできないことが、どんなに多いか、数えてみることである。
そうすれば、「友人は第二の自我である(†22)」と言っているのは、古代の人々の控え目な言い方であったことが
照明らかになるであろう。
友人ははるかに自分以上の者だからである。
人々には寿命があり、しばしばひどく気にかけていることを、あれこれ願いながら亡くなる。
子供を結婚させること、事業を完成させること、その他この種のことである。
真の友人をもっているならば、自分の死後にもこうしたことの世話がつづけられることを、ほとんど確信してよい。
それゆえ、人間は欲望の中にいわば二つの生命をもっている。
彼は身体をもっており、その身体は一つの場所に閉じこめられている。
しかし友情がある場合には、生涯のすべての役目はいわば彼とその代理人に委ねられる。
彼はそれらの役目を友人によって果たすことができるからである。
人間には、どんな顔または容姿をしていても、自分ひとりで言ったりしたりするわけにはいかないことが、
どんなに多いことであろうか。
自分自身の功績を控え目に申し立てることも、まして賞賛することは、ほとんどできない。
時には嘆願したり懇願したりすることに堪えられないし、またこの種のことは多い。
しかし、すべてこうしたことは友人の口にかかると優雅であり、自分自身の口から出ると恥知らずである。
さらに人の身の上には多くの独得な関係があって、それらを取り除くことはできない。
息子には父としてしか、妻には夫としてしか、敵には条件つきでしか話しかけることができない。
ところが、友人はその時の事情に応じて、こちらの身の上のことなどおかまいなく、話すことができる。
しかし、こうしたことを数え上げたら、切りがないだろう。
私が述べたのは、人が自分自身の役割をうまく演じられない場合の通則である。
友人がなければ、舞台を去ってもさしつかえない。
